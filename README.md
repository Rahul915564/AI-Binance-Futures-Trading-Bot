# 📈 AI Trading Strategy

The bot uses:
- RSI (Relative Strength Index)
- EMA20 Trend Analysis

### BUY Logic
- RSI < 30
- Price Above EMA20

### SELL Logic
- RSI > 70
- Price Below EMA20
